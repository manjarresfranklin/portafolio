Portafolio de Ciencia de Datos

Proyectos aplicados en regresión, clasificación y procesamiento de lenguaje natural usando Python, R y Deep Learning.

Detección de Fraude

Se desarrolló un modelo con árboles de decisión y bosques aleatorios para detectar fraudes en canastas de compra. Se optimizaron los umbrales de clasificación para mejorar la sensibilidad, aunque con cierto costo en falsos positivos. El modelo final logró una exactitud del 89.97%. Se aplicó análisis de importancia de variables y limpieza masiva de datos.

Predicción de Precios

Se predijeron precios inmobiliarios usando regresión robusta (Huber) tras una cuidadosa selección de variables. Se imputaron valores faltantes y se detectaron outliers con impacto significativo. El modelo alcanzó un MAE de 169,465, destacando la influencia de variables como número de habitaciones y tipo de propiedad. Se garantizó interpretabilidad y robustez.

Reseñas con LSTM

Se entrenó una red neuronal LSTM con embeddings preentrenados para clasificar reseñas de hoteles como positivas o negativas. Se realizó preprocesamiento exhaustivo del texto y se optimizó la arquitectura para mejorar el rendimiento. El modelo fue evaluado con métricas como precisión y F1-score. Este proyecto aplica PLN y Deep Learning en texto real.
